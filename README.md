# dotnet-vue_flickr-api
Proxy API in .NET Core that allows you to view / search images in VUE frontend with the Flickr API

## Installation / Run

### Backend
In the root folder run: 

`dotnet run`

### Frontend
In `ROOT/Client/pixplorer` run:

`npm install`

`npm run serve`

## Web Browser
Navigate to the following: 

`http://localhost:8080/#`
